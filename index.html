My process usually begins with pen and paper, but right after that - I use Figma, perfect for designing and prototyping. If I need a custom element, then I’ll choose to design it in Illustrator.

Animations are usualy done in Framer Motion since it is compatible with React, but I find myself using GSAP more and more.

React is my JavaScript library of choice and I use Redux for state managment.

Since I use JavaScript it is natural that I am going to use it on the backend with NodeJS.

The first database I use depends on the project, but it is usually MongoDB or MySQL.
